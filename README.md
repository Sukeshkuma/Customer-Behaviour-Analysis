**Customer Shopping Behavior Analysis:**

**Project Overview:-**

**This project focuses on analyzing customer shopping behavior and purchasing patterns using transactional data from over 40,000 purchases across various products, categories, and age groups. The primary objective is to transform raw transactional data into a structured, reliable format and extract meaningful insights that can support data-driven business decisions.**

**The analysis helps identify customer preferences, purchasing trends, and category-level performance, which can be leveraged for strategic planning and operational improvements.**

**Objectives:-**

Clean and preprocess raw transactional data

Organize and manage data in a structured format for future use

Analyze customer behavior and purchasing patterns

Answer business-driven questions using SQL

Create insightful and interactive dashboards for visualization

Derive actionable insights to support strategic decision-making

**Tools & Technologies Used:-**

**Python**

Data cleaning and transformation

Handling missing values, duplicates, and data inconsistencies

Exploratory Data Analysis (EDA)

**SQL**

Writing queries to answer business questions.

Aggregations, filtering, joins,windows and analytical functions.

**Power BI**

Data visualization and dashboard creation

Interactive reports to present key insights clearly

**Key Insights:-**

Customer purchasing trends across different age groups

High-performing products and categories

Shopping frequency and behavior patterns

Data-backed insights to guide business strategies
